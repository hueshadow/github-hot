## 简介

Vite 在开发与生产模式下采用不同策略：开发时利用浏览器原生 ESM，生产侧基于 Rollup 打包，体验与生态都非常成熟。

## 为什么常出现在「热门」里

- 插件与模板丰富（Vue / React / Svelte 等）
- 与 Vitest、SSR 框架（如 Astro）配合紧密
